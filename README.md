
# Customer Shopping Behavior Dashboard
An end-to-end Data Analytics project focused on understanding customer shopping behavior using Python, MySQL, and Power BI.

# Project Overview
This project analyzes customer shopping behavior using transactional retail data from 3,900 customer purchases across multiple product categories.

The objective is to uncover:
- Customer spending patterns
- Revenue-driving customer segments
- Product performance
- Subscription behavior
- Shipping preferences
- Business growth opportunities

The project combines:
- Python for data cleaning & preprocessing
- MySQL for business analysis
- Power BI for interactive dashboard visualization

# Key Insights

- Non-subscribers contribute 73% of customers
- Male customers generate higher revenue
- Clothing category contributes the highest revenue
- Young adults are the top revenue contributors
- Free shipping generates the highest revenue
- Avg customer spend is approximately $59.76
- Total revenue generated exceeds $233K
